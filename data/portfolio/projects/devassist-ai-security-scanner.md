# DevAssist AI - RAG Platform and React Security Scanner

## Summary
DevAssist AI is a full-stack Retrieval-Augmented Generation (RAG) platform that uses OpenAI APIs to deliver context-aware Q&A, summarization, code explanation, and source-backed AI responses. The project also includes AI-assisted security tooling for React applications.

## Problem
Developers need AI assistance that can reason over project-specific documents and source material without producing ungrounded responses. React applications also benefit from repeatable security analysis that is faster and more consistent than manual review alone.

## Responsibilities
- Built the full-stack AI application
- Implemented Retrieval-Augmented Generation workflows
- Designed document chunking and semantic search behavior
- Integrated OpenAI embeddings and cosine similarity retrieval
- Developed chat, summarization, and code explanation experiences
- Added conversation history and cited sources for AI responses
- Designed the React security scanner CLI architecture
- Built configuration, scanner, and entrypoint modules for security analysis
- Integrated OpenAI APIs
- Structured scanner output as machine-readable vulnerability reports
- Applied the tool within AI-assisted development workflows
